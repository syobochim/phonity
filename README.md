# Phonity

## Development

### Requirement

* [Grunt: The JavaScript Task Runner](http://gruntjs.com/)
```
$ npm install -g grunt-cli
```

* [Bower - A package manager for the web](http://bower.io/)
```
$ npm install -g bower
```

### Install libraries

```
$ git clone https://github.com/tenten0213/phonity.git
$ cd phonity
$ grunt install
$ bower install
```
